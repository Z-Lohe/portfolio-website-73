# \# Full-Stack Portfolio Website



This is a full-stack portfolio website developed as part of the  

Web Development Workflow (SEC) course.



The project demonstrates frontend development, backend integration,  

database connectivity, and deployment workflow.



## \## 🚀 Technologies Used



#### \### Frontend

* HTML
* CSS
* JavaScript



#### \### Backend

* Node.js
* Express.js



#### \### Database

* MongoDB Atlas



#### \### Version Control \& Deployment

* Git
* GitHub
* GitHub Pages



## \## ✨ Features



* Creative and interactive UI
* Floating animated shapes and micro-interactions
* Responsive design
* Contact form with backend integration
* Messages stored in MongoDB database
* Hosted frontend using GitHub Pages
* Full-stack project structure

## \## ⚙️ How to Run Backend Locally



* Open terminal
* Navigate to backend folder: cd backend
* Install dependencies: npm install
* Start server: node server.js



Server runs at:

http://localhost:3000
